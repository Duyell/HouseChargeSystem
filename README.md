# 房地产项目管理系统
## 项目简介
### 基于SpringBoot与MyBatis,使用HTML与CSS与JS建立了一个在本机上运行的房地产管理项目,项目功能介绍:
### 1.	查询广场功能：支持用户按房产类型、阳台方向、价格范围、建筑面积、使用面积、楼层范围等多条件组合筛选房产；筛选结果以表格形式展示，提供户型详细介绍入口，点击可查看房产编号、户型图、价格等完整信息，还包含有预订和取消预订功能。
### 2.	房产看台功能：展示楼盘基础信息（位置、特点、楼盘内的房产的预订情况等）；实时更新房源预订状态，明确标注已预订房源数量及占比，帮助用户快速了解楼盘供需情况。
### 3.	预订功能：用户选择可预订房产后，提交预订申请，系统校验用户身份与房源状态，通过后生成唯一预订单号，同步更新房产预订状态，并向用户反馈预订成功回执。
### 4.	取消预订功能：用户提交取消预订申请，系统校验预订单号与用户身份，通过后更新房产状态为可预订，记录取消时间，并反馈取消成功回执。
### 5.	公司管理功能：管理人员可录入楼盘信息（名称、地址、特点等）和户型信息（房产类型、面积、阳台方向、户型图、价格等），系统支持数据验证与重复校验，确保录入信息准确有效。
### 6.	房源状态编辑功能：管理人员可手动更新房产状态，系统记录状态变更日志，反馈更新结果，保障房源信息实时准确。
